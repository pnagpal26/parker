# RECO Attribution: Confirmed Wording

**Status:** CONFIRMED
**Date confirmed:** 2026-03-15
**Source:** RECO Bulletins 5.1 + 5.3 (January 2024, TRESA)
**Requirement ID:** PRE-03

## Confirmed Attribution String

This exact string must appear on every ad — no abbreviation, no substitution:

```
Garima Nagpal · Team Nagpal · Sales Representative · RE/MAX Hallmark Realty, Brokerage
```

## Why Each Element Is Required

| Element | Requirement | Source |
|---------|------------|--------|
| "Garima Nagpal" | Registrant's name as registered with RECO | Bulletin 5.1 |
| "Team Nagpal" | Optional — included for brand recognition; does not substitute for brokerage name | Bulletin 5.1 |
| "Sales Representative" | Registrant's permitted title as registered with RECO | Bulletin 5.1 |
| "RE/MAX Hallmark Realty" | Brokerage trade name as registered with RECO | Bulletin 5.1 |
| ", Brokerage" | Required designator — "brokerage" or "real estate brokerage" must accompany the name | Bulletin 5.1 |

**CRITICAL:** "Team Nagpal" alone does NOT satisfy the brokerage disclosure requirement.
"RE/MAX Hallmark Realty, Brokerage" must be present and no smaller than "Garima Nagpal".

## Figma Text Properties

Apply these properties to the attribution text layer in every ad:

- **Font:** Plus Jakarta Sans, weight 400
- **Size:** 18–20px (at 1080px canvas width)
- **Color:** rgba(255, 255, 255, 0.5)
- **Letter spacing:** 0.15em
- **Placement:** y=1480–1520px (bottom of safe zone, above y=1536 safe line)
- **Width:** Full safe zone width (x=90 to x=990)
- **Alignment:** Left-aligned

## Compliance Notes

- RECO standard is "clearly and prominently" displayed — test by covering all other
  elements and confirming the attribution line is still readable
- No minimum point size specified by RECO; 18–20px at 1080px canvas reads at
  approximately 14–15pt on a phone screen — within standard legibility thresholds
- If background makes white text hard to read, add a semi-transparent backing panel
  (rgba(0,0,0,0.3)) behind the attribution zone

## Registration Note

Attribution wording assumes Garima is registered with RECO as "Sales Representative"
(the more common status). If Garima holds a broker's license, replace "Sales
Representative" with "Broker" in the confirmed string above. Verify at reco.on.ca
if uncertain.

Every Phase 2 ad PLAN.md must reference this file and copy the attribution string
verbatim. Do not paraphrase or abbreviate on any ad.
